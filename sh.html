Software

Perangkat lunak (Software) merupakan program-program yang ada pada komputer yang digunakan untuk menjalankan tugas atau perintah tertentu yang diinginkan oleh pengguna komputer.

Operating System

  1. Menjalankan operasi dasar
  2. Mengatur kerja hardware dan software
  3. Wadah program atau aplikasi

Apps

  1. Utility
  2. Multimedia
  3. Office/Programme
  4. Internet & security

Hardware

Hardware (Perangkat Keras) adalah salah satu komponen dari sebuah komputer yang sifat alat nya bisa dilihat, diraba, dan dirasakan oleh manusia secara langsung atau yang berbentuk nyata, yang berfungsi untuk mendukung proses komputerisasi.

Definisi Hardware penyimpanan, tempat penyimpanan, media yang digunakan untuk menyimpan data yang diolah oleh komputer.

3 Komponen utama dalam Hardware :

  1. Input
  2. Proces
  3. Output

3 Tipe Storage :

  1. SSD
  2. HDD
  3. SSHD

Manajemen & Sequrity File

Apa sih itu file?

File adalah abstraksi penyimpanan dan pengambilan informasi di disk. Abstraksi ini membuat user tidak dibebani rincian cara dan letak penyimpanan data.

Fungsi Management File :

  1. Memudahkan cara kerja atau mekanisme pemakaian file secara bersamaan.
  2. Kemudahan dalam membuat, mengedit dan menghapus file.
  3. Kemudahan dalam melakukan Backup dan Recovery untuk antisipasi kehilangan file kecelakaan atau upaya orang lain untuk merusak atau menghancurkan file.
  4. Dapat disimpan dengan Aman dan Rahasia pada lingkungan yang sensitif.

Contoh file security :

Password File

Cara memberikan password pada file :

  1. Pilih file yang ingin diamankan.
  2. Klik File > Save As.
  3. Klik Tools (ada di sebelah kiri tombol Save) > General Option.
  4. Masukkan password yang ingin digunakan.
  5. Klik Ok.
  6. Masukkan kembali password untuk membuka dan mengedit file.
  7. Klik Ok.
  8. Selesai.

Hidden file

Cara memberi atribut hidden pada file :

  1. Klik kanan pada file/folder kemudian pilih Properties.
  2. Pada Attributes, centang checkbox Hidden.
  3. Pada menu bar File Manager, klik View kemudian pilih Show hidden Files.

Zip File

Cara membuat file ke zip :

  1. Klik kanan pada Folder kemudian pilih Add to archive.
  2. Beri nama dan tipe archive format.
  3. Pilih Set Password kemudian enter password lalu klik Ok.
